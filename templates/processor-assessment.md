# Third-Party Processor / Vendor Assessment Questionnaire

Use this questionnaire before engaging a data processor (sub-processor) and annually thereafter.
Required under GDPR Art. 28 ("controller shall only use processors providing sufficient guarantees").

|---|---|---|---|
| 2.1 | Is the vendor willing to sign a Data Processing Agreement (GDPR Art. 28)? | Yes / No | |
| 2.2 | Does the DPA cover the required Art. 28(3) clauses? | Yes / No | Review against checklist below |
| 2.3 | If EU SCCs are needed for data transfers, will the vendor execute them? | Yes / No / N/A | |
| 2.4 | Does the vendor's DPA cover LGPD Art. 37 obligations? | Yes / No / N/A | (if LGPD applies) |

**GDPR Art. 28(3) DPA checklist:**
- [ ] Process only on documented instructions
- [ ] Confidentiality obligations on authorised staff
- [ ] Technical and organisational security measures (Art. 32)
- [ ] Sub-processor approval and flow-down obligations
- [ ] Assistance with data subject rights requests
- [ ] Assistance with breach notifications (Art. 33/34)
- [ ] Deletion or return of data at end of service
- [ ] Provision of all necessary information / audit rights

# Data Subject Access Request (DSAR) — Response Templates

Response letter templates for the most common data subject rights under GDPR (Art. 15–22), LGPD (Art. 18), and CCPA (§1798.100–.125).

## Internal DSAR Process Checklist

- [ ] Request received and logged (include date, channel, request type)
- [ ] Identity verified (before sharing any data)
- [ ] Request classified (access / erasure / restriction / portability / correction / objection)
- [ ] Clock started (GDPR: 1 month; CCPA: 45 days; LGPD: 15 days)
- [ ] Data inventory queried (all systems containing data subject's records)
- [ ] Third-party processors notified if erasure/restriction required
- [ ] Response drafted and reviewed by DPO/Legal
- [ ] Response sent via secure channel
- [ ] Request closed and outcome logged
- [ ] If extension needed: notified data subject within original deadline
